## 📖 Sobre

Este repositório reúne as atividades, projetos e materiais desenvolvidos ao longo do **curso de DevOps da Aponti**.

O objetivo é registrar minha evolução durante a formação, documentando tanto os conceitos estudados quanto as práticas aplicadas utilizando ferramentas e metodologias amplamente utilizadas no mercado.

---

## 🎯 Objetivos

- Consolidar os conhecimentos adquiridos durante o curso;
- Aplicar boas práticas de versionamento utilizando Git e GitHub;
- Desenvolver projetos seguindo conceitos da cultura DevOps;
- Organizar todas as atividades em um único repositório;
- Documentar o aprendizado para futuras consultas.

---

## 🛠️ Tecnologias e Ferramentas

Ao longo do curso serão utilizadas tecnologias como:

- Git
- GitHub
- Markdown

---

## 📈 Objetivo deste repositório

Este repositório funciona como um registro da minha evolução durante o curso, reunindo atividades práticas, projetos e documentações produzidas ao longo da formação.

Sempre que uma atividade for realizada em equipe, o respectivo README conterá um link para o repositório original do grupo, preservando o histórico completo de contribuições e commits.
