## 📖 Sobre

O objetivo desta atividade foi planejar o desenvolvimento de um sistema utilizando uma metodologia ágil (**Scrum** ou **Kanban**), organizando as atividades necessárias para a futura construção de uma pipeline de CI/CD.

A atividade foi desenvolvida em equipe, simulando o planejamento inicial de um projeto de software, incluindo a definição da metodologia de gerenciamento, arquitetura da aplicação, backlog, organização das tarefas, prioridades e planejamento da pipeline.

---

## 🏛️ Cenário

### Sistema para Biblioteca Municipal

O sistema proposto possui as seguintes funcionalidades:

- Cadastro de livros;
- Empréstimos;
- Devoluções;
- Cadastro de usuários;
- Relatórios.

---

## 🎯 Objetivos da atividade

- Definir uma metodologia ágil adequada ao projeto;
- Planejar a arquitetura do sistema;
- Organizar o backlog;
- Definir prioridades de desenvolvimento;
- Distribuir responsabilidades entre os integrantes da equipe;
- Planejar uma futura pipeline de CI/CD;
- Elaborar uma proposta para apresentação ao cliente.

---

## 📋 Requisitos desenvolvidos

Durante a atividade foram definidos:

- Metodologia ágil e justificativa da escolha;
- Backlog do produto;
- Arquitetura do sistema;
- Organização das tarefas;
- Priorização das funcionalidades;
- Distribuição das responsabilidades;
- Planejamento da pipeline de CI/CD;
- Documentação da proposta.
